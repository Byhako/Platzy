## Conectar proyecto con firebase.

Primero creamos poryecto desde la consola de firebase, luego:

$ sudo npm install -g firebase-tools
$ firebase login
$ firebase init
$ firebase use --add
$ firebase serve

+ En archivo firebase.json agregar:

```
  "hosting": {
    "public": "./"
  }
```
